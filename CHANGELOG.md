# Changelog

All notable changes to VirusFor.ai will be documented in this file.

## [1.0.0] — 2026-03-19

### Added
- SmartContract marketplace with community ratings and category filtering
- Multi-step workflow execution with email-based user assignment
- Real-time chat via WebSocket (Socket.io) with user presence
- Friend system and community groups
- Electron desktop application with offline sync capability
- License key authentication with plan-based access control
- Full audit logging at every workflow step
- Notification system for workflow events and assignments
